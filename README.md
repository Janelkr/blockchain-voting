# 🧩 Blockchain Voting DApp

Decentralized voting application using Ethereum smart contracts, Hardhat, and Ethers.js.  
Users can vote for candidates via MetaMask, and the results are stored immutably on the blockchain.

## ⚙️ Technologies

- **Solidity** – smart contract  
- **Hardhat** – development & deployment  
- **Ethers.js** – blockchain interaction  
- **MetaMask** – wallet & transaction signing  
- **HTML/CSS/JS** – frontend (single file)
